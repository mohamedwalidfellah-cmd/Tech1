# Tech1 - AutoDiag Application

Application web mobile-first complète en français pour un service de diagnostic et réparation électrique automobile à domicile en Algérie.

## Modules

1. **Module Client** - Parcours diagnostic interactif
2. **Module Admin** - Dashboard de gestion et dispatch
3. **Module Technicien** - Interface terrain mobile

## Installation

```bash
npm install
npm run dev
```

## Zones Couvertes

- Alger Centre
- Alger Ouest
- Blida
- Tipaza

## Design

Couleurs : Noir (#1a1a1a), Jaune Énergie (#FFD700), Gris Foncé (#2d2d2d)
